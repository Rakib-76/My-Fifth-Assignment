Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML Gets or sets the HTML content of an element, including HTML tags, Inner-Text Gets or sets the visible text content of an element,TextContent Gets or sets the text content of an element, including all elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element, instead of adding listeners to multiple child elements individually.

Question-4: What is event bubbling in the DOM?

Event bubbling is a phase in the DOM event flow where an event starts at the target element and bubbles up the DOM tree to its parent, grandparent.

Question-5: How do you create, add, and remove elements using JavaScript?

Create Element :

Use the document.createElement() method to create new elements.

Add Element :

Use the document.body.appendChild() method to add new elements.

Remove Element :

Use the container.removeChild() method to remove a child element.
